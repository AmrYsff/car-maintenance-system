# Car Maintenance and Repair Center Management System

## Project Overview
A Java-based system designed to manage customers, vehicles, services, and maintenance operations
in a Car Maintenance and Repair Center.

## Technologies
- Java
- Object-Oriented Programming
- JUnit (Unit Testing)

## Features
- Customer and vehicle management
- Service scheduling and tracking
- Mechanic assignment
- Maintenance history records

## Testing
Unit testing implemented using JUnit.
